MyBatis Plus 执行原生SQL异常 Failed to process、net.sf.jsqlparser.parser.ParseException: Encountered unexpected token

发表于 2021-06-19 更新于 2022-05-19 分类于 MyBatisPlus

SpringBoot项目在执行原生SQL（使用IPage）时异常：net.sf.jsqlparser.parser.ParseException: Encountered unexpected token:XXXXX

总结

  1. MyBatis-Plus在处理SQL时会被 MybatisPlusInterceptor 拦截，对SQL进行一系列操作，如若不需要使用注解 @InterceptorIgnore 忽略拦截即可。
  2. MyBatis-Plus在处理分页countSql时，会经过PaginationInnerInterceptor.java拦截器，在获取执行SQL时会通过，Page类中optimizeCountSql属性判断是都需要优化，如果不需要再传Page的时候设置为false即可。
