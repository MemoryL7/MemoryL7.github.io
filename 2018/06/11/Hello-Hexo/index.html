Hello Hexo

Jun 11 2018

GitHub + Hexo
整体环境：Mac OS

Hexo ?

Hexo是一个快速、简洁高效的博客框架，主要通过Makedown解析文章。

如果你也想拥有自己的博客，就一起来创建吧💪

本来是想在CentOS下安装的，无奈升级GCC一直失败😌

以下安装流程在Mac OS下安装运行，Window与其大致相同可做参考😊

安装前准备

Node.js、Git 检查

检测本机是否安装Node.js,Git

$ node -v
$ npm -v

$ git --version

如果没有安装，macOS 和 windows下支持通过应用程序安装的，所以下载后一键安装都没什么问题。

GitHub账户创建和新建Repository(项目)

如果自己已经有账号，直接创建项目，否则需要自己先创建一个GitHub账号

新建 Repository

需要注意一下规范
1. Repository Name(项目名称)
   博客名称.github.io
2. 勾选Initialize this repository with a README

创建Repository

需要图片

Settings

Settings图片

Hexo 安装

本地创建存放Blog博客的文件夹

在自己认为合适的地方创建文件夹即可

Mac OS

$ mkdir ~/blog
$ cd ~/blog

Windows

$ d:
$ mkdir blog
$ cd blog

通过npm安装Hexo，如果太慢的话可以了解一下 cnpm 😜

$ npm install hexo -g

安装完成后，检查是否安装成功

$ hexo -v

初始化 Hexo

$ hexo init

稍微等待之后 “Start blogging with Hexo” 🎉🎉🎉

安装 Hexo 相关组件

$ npm install

完成后现在可以在本地体验 Hexo

$ hexo g
$ hexo s

页面如果访问先停止服务Command+C或者Ctrl+C，之后指定端口号启动

$ hexo server -p 6666

到目前为止本地已经可以访问了，下来就需要关联GitHub发布到外网🆙
